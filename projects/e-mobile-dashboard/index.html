e-mobile.lv dashboard

design and development

e-mobile.lv is one of the leading used cars sellers in the Baltics. To make sure the company has the right tools for the job we created a dashboard that helps the team with everyday tasks and clear communication between co-workers.

Dashboard is designed so that each employee can easily work with upcoming tasks, client database and archive, as well as, integrated tools like costs calculator. Management can track website visits, number of deals in progress, costs and other vital information.
